## 📌 개요

## 🔐 관리자 접속

- **URL**: [관리자 페이지 URL]
- **권한**: ADMIN 역할 필요

## 👥 사용자 관리

### 사용자 목록 조회

1. **관리** → **사용자 관리** 메뉴
2. 검색/필터 기능으로 사용자 조회

### 사용자 권한 변경

1. 사용자 상세 페이지 접속
2. **권한** 섹션에서 역할 선택
3. **저장** 클릭

### 권한 체계

| 역할   | 권한        |
| ------ | ----------- |
| ADMIN  | 전체 관리   |
| EDITOR | 콘텐츠 관리 |
| USER   | 읽기 전용   |
