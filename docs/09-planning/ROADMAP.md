## 📅 한깨봄 2.0 로드맵

### 1단계: 기반 구축 및 생태계 런칭 (현재 완료 ✅)
**핵심**: 교육 + 프리랜서(Stars) + 관리자(Admin) 통합 운영
*   [x] **Core**: Next.js 15 + NestJS 11 모노레포 아키텍처 구축
*   [x] **Education**: 1기 설명회 랜딩 및 교육 정보 페이지
*   [x] **Stars**: 프리랜서 마이페이지, 프로젝트 보드, 정산 시스템 (1/2차)
*   [x] **Contests**: 공모전 접수 및 메인 페이지
*   [x] **Admin**: 통합 대시보드 및 모듈별(Stars/Edu/Contest) 관리 도구
*   [x] **Infra**: Cloud Run, Supabase, Cloudflare R2/Stream 배포 완료

### 2단계: 비즈니스 확장 (2026.Q1 ~ Q2)
**핵심**: 수익 모델 다각화 (AI 스튜디오 + 마케팅 대행)
*   [ ] **AI Studio**: B2B 클라이언트 전용 의뢰 및 제작 파이프라인
*   [ ] **Marketing**: 대행업체 전용 마케팅 요청 및 성과 리포트
*   [ ] **Commerce**: 전자결제(PG) 연동 및 인보이스 자동화

### 3단계: 생태계 고도화 (2026.Q3 ~)
**핵심**: 데이터 기반 자동화 및 상담사 네트워크
*   [ ] **Counselor System**: 500명 상담사 전용 파트너 센터
*   [ ] **AI Matching**: 프로젝트-프리랜서 자동 매칭 알고리즘
*   [ ] **Global**: 다국어 지원 (영문/일문)
