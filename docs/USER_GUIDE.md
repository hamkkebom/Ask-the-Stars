## 🚀 시작하기

### 서비스 접속
- 웹 브라우저에서 [서비스 URL]에 접속
- 지원 브라우저: Chrome, Firefox, Safari, Edge

### 회원가입
1. **회원가입** 버튼 클릭
2. 이메일, 비밀번호, 이름 입력
3. 이메일 인증 완료

### 로그인
1. 이메일/비밀번호 입력
2. 또는 소셜 로그인 (Google, Kakao)

## 📋 주요 기능

### 대시보드
- 최근 활동 요약
- 빠른 작업 버튼
- 알림 목록

### 콘텐츠 관리
| 기능 | 설명 |
|------|------|
| 작성 | 새 글 작성 |
| 수정 | 기존 글 편집 |
| 삭제 | 글 삭제 (복구 불가) |

### 프로필 관리
- 정보 수정
- 비밀번호 변경
- 알림 설정
