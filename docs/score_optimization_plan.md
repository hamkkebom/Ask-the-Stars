## 🏗️ Phase 2: 핵심 기능 완성 (1-2주차)

> **"보완 필요 사항" 중 기능적 누락 채우기**
> 사용자가 기대하는 동작(필터, 정렬, 다운로드)을 완성합니다.

### 1. 필터링 및 검색 강화
- **공통**: 각 목록 페이지(상담사, 공모전, 요청 등)에 '데이터 없음' 상태 UI 통일.
- **`/counselors`**: 전문 분야별 필터링 기능 실제 동작 확인 및 고도화.
- **`/moon/management/freelancers`**: 프리랜서 등급/상태별 필터링 구현.

### 2. 데이터 처리 및 연동
- **`/stars/earnings`**: 정산 명세서 및 목록 **PDF/Excel 다운로드** 기능 구현.
- **`/studio/request`**: 파일 업로드 UI 개선(드래그앤드롭, 진행률 표시) 및 유효성 검사.
- **`/moon/management`**: 대량 메시지 발송, 요청 복제 등 관리자 편의 기능 추가.

### 3. 상호작용 및 피드백
- **`/education/waitlist`**: 등록 완료 후 명확한 성공 메시지(Toast/Modal) 및 홈으로 이동/추가 탐색 유도.
- **`/counselor/dashboard`**: 미답변 질문 알림 배지, 오늘의 일정 하이라이팅.

## 🔍 Phase 4: SEO & 접근성 & 성능 (4주차)

> **마지막 2점 채우기 (98점 → 100점)**
> 눈에 보이지 않지만 품질을 결정하는 기술적 요소입니다.

### 1. 검색 엔진 최적화 (SEO)
- **JSON-LD 구조화 데이터 적용**:
    - `/videos/*`: VideoObject 스키마
    - `/news/*`: NewsArticle 스키마
    - `/education/courses/*`: Course/Product 스키마
    - `/about/contact`: LocalBusiness 스키마
- **메타태그 보완**: 모든 페이지의 `title`, `description`, `og:image` 전수 검사 및 고유값 적용.
- **sitemap.xml**: 동적 라우트(뉴스, 영상, 공모전 등) 포함하여 완벽 생성.

### 2. 웹 접근성 (A11y)
- **Alt 텍스트**: 모든 이미지(특히 `/studio/creators` 등)에 의미 있는 대체 텍스트 자동/수동 적용.
- **키보드 접근성**: 탭 키로 모든 메뉴 및 버튼 접근 가능하도록 `tabIndex` 및 포커스 스타일 점검.

### 3. 성능 최적화
- **이미지**: 차세대 포맷(WebP/AVIF) 적용 및 레이지 로딩(Lazy Loading) 전면 적용.
- **번들링**: 불필요한 라이브러리 제거 및 코드 스플리팅으로 초기 로딩 속도(LCP) 1.2초 로 단축.
