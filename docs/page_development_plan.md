# 📋 페이지 개발 구현 계획

> **작성일**: 2026-01-21  
> **예상 기간**: 5주  
> **기반 문서**: [site_map_audit.md](./site_map_audit.md)

## 🎯 목표

74개 페이지 중:

- ❌ **미구현 3개** → 신규 생성
- 📝 **기획됨 10개** → 개발중으로 업그레이드
- 🔨 **개발중 52개** → 완성으로 업그레이드

---

## Phase 1: 미구현 필수 페이지 (1주차)

> **목표**: 핵심 사용자 흐름에 필요한 누락 페이지 생성

### 신규 생성 페이지

| 경로 | 페이지명 | 기능 |
|------|---------|------|
| `/requests` | 견적 요청 목록 | 요청 목록, 필터링, 새 요청 작성 |
| `/requests/[id]` | 요청 상세 | 상세 정보, 견적 응답, 메시지 |
| `/dashboard` | 통합 대시보드 | 역할별 대시보드 리다이렉트 |

---

## Phase 2: 기획됨 페이지 완성 (2주차)

> **목표**: 기본 구조만 있는 페이지에 실제 기능 추가

| 섹션 | 페이지 | 작업 내용 |
|------|--------|----------|
| 공개 | `/contests` | 공모전 목록, 상세, 참가 신청 |
| Stars | `/stars/resources` | 리소스 라이브러리, 다운로드 |
| Moon | `/moon/feedback/templates` | 템플릿 CRUD, 카테고리 |
| Moon | `/moon/advertising/campaigns` | 캠페인 생성/관리 |
| Moon | `/moon/advertising/analytics` | 광고 성과 차트 |
| Admin | `/admin/studio` | 스튜디오 설정 관리 |
| Admin | `/admin/contests` | 공모전 생성/관리 |
| Admin | `/admin/tasks` | 작업 대기열 관리 |

---

## Phase 3: 개발중 페이지 완성도 향상 (3-4주차)

> **목표**: 70점대 페이지를 80점 이상으로 업그레이드

### 우선순위별 작업

| 우선순위 | 대상 | 작업 |
|----------|------|------|
| 높음 | 인증 페이지 8개 | 폼 검증, 에러 메시지, 이메일 연동 |
| 높음 | Stars 핵심 기능 | 프로젝트 상세, 피드백, 포트폴리오 |
| 중간 | Moon 관리 기능 | 테이블 필터링, 정산 API 연동 |
| 중간 | Counselor 전체 | 통일된 UI, API 연동 |
| 낮음 | Admin 기타 | CRUD 완료, 차트 완성 |

## 검증 계획

### 자동화 테스트

```bash
pnpm build   # 빌드 테스트
pnpm lint    # 린트 검사
```

### 수동 테스트

1. 각 페이지 접근 가능 여부 확인
2. 반응형 (모바일/태블릿/데스크톱) 확인
3. 사용자 흐름 테스트 (가입→로그인→대시보드)
