# 🗺️ 별들에게 물어봐 (Ask the Stars) - 전체 사이트맵

## 📊 사이트 구조 개요

```
사주천궁 × 별들에게 물어봐
├── 🔐 인증 시스템
├── 🌙 달님(Moon) 전용 페이지
├── ⭐ 별님(Stars) 전용 페이지
├── 🔮 상담사(Counselor) 페이지
└── 🌐 공개 페이지
```

---

## 🔐 1. 인증 시스템

```
/auth
├── /login                    # 로그인
├── /register                 # 회원가입
│   ├── /moon                # 달님 가입 (초대 코드 필요)
│   ├── /stars               # 별님 가입 (심사 필요)
│   └── /counselor           # 상담사 가입
├── /forgot-password         # 비밀번호 찾기
└── /verify-email            # 이메일 인증
```
