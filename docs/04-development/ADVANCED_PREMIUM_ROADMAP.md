# Advanced Premium Roadmap: "Beyond Playback"

## 🚀 목표 (Goal)
단순한 "재생(Playback)"을 넘어, 넷플릭스/유튜브 수준의 **몰입감(Immersion)**과 **지능형 기능(Intelligence)**을 제공하는 시스템으로 진화합니다.

---

## 1. 사용자 경험 혁신 (UX Innovation)

### A. 스마트 이어보기 (Cross-Device Resume)
*   **현재**: 새로고침하면 처음부터 재생됨.
*   **개선 (Better)**:
    *   플레이어가 10초마다 시청 위치(`timestamp`)를 서버에 전송 (Heartbeat).
    *   PC에서 보던 영상을 모바일에서 켰을 때 **"1분 20초부터 이어보시겠습니까?"** 팝업 제공.
    *   **구현 기술**: `Redis` (빠른 쓰기) -> `PostgreSQL` (영구 저장).

### B. 인스턴트 프리뷰 (Instant Preview)
*   **현재**: 썸네일 이미지만 정적으로 표시.
*   **개선 (Better)**:
    *   마우스를 올리면(Hover) 소리 없이 **가장 흥미로운 3초 구간**이 자동 재생.
    *   Cloudflare Stream의 `GIF/WebP` 생성 API를 활용하여 트래픽 소모 없이 구현.

### C. 딥 서치 (Deep Content Search)
*   **현재**: 제목과 설명만 검색 가능.
*   **개선 (Better)**:
    *   영상 내 **"대사(Speech)"**를 검색.
    *   예: "리액트 훅" 검색 -> "3분 40초: 자, 이제 리액트 훅에 대해..." (해당 위치로 바로 이동).
    *   **구현**: Cloudflare AI Caption -> VTT 다운로드 -> DB 텍스트 인덱싱.

---

## 2. 보안 및 수익화 강화 (Security & Monetization)

### A. 다이내믹 워터마크 (Dynamic Watermarking)
*   **현재**: 다운로드 방지만 가능. (화면 녹화는 못 막음)
*   **개선 (Better)**:
    *   영상 재생 시 시청자의 **"이메일/ID"가 희미하게 오버레이** 됨.
    *   화면 녹화 유출 시 누구 계정인지 즉시 추적 가능. (유료 강의 보안 필수)

### B. 단기 토큰 로테이션 (Short-Lived Token Rotation)
*   **현재**: 1~2시간짜리 토큰 발급.
*   **개선 (Better)**:
    *   **5분 만료 토큰** 발급 + 백그라운드 자동 갱신.
    *   토큰을 친구에게 공유해도 5분 뒤면 즉시 차단됨.

---

## 3. 운영 효율화 (DevOps)

### A. TUS 리주머블 업로드 (Resumable Upload)
*   **현재**: 업로드 중 인터넷 끊기면 실패.
*   **개선 (Better)**:
    *   10GB 파일도 **이어올리기(Resume)** 지원.
    *   와이파이가 끊겨도 재연결 시 자동으로 90%부터 업로드 재개.

### B. 자동 품질 검수 (Quality Gate)
*   **현재**: 업로드되면 바로 게시.
*   **개선 (Better)**:
    *   업로드 직후 오디오 레벨(Loudness) 분석 -> "소리가 너무 작습니다" 경고 알림.
    *   블랙 프레임 감지 등을 통해 품질 미달 영상 자동 필터링.

---

## 📅 단계별 도입 추천
1.  **Phase 1 (필수)**: 이어보기(Resume), TUS 업로드. (사용자 불만 제거)
2.  **Phase 2 (차별화)**: 딥 서치, 인스턴트 프리뷰. (와우 포인트)
3.  **Phase 3 (보안)**: 워터마크, 토큰 로테이션. (수익 보호)
